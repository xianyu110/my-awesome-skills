# 公众号自动化写作工作流

一个结构化、透明化、真实化的AI辅助写作系统。

## 特点

- **两层判断机制**：工作区判断 + 任务类型判断，避免AI误判
- **9步完整流程**：从需求理解到配图完成，系统化协作
- **三遍审校机制**：内容审校 → 风格审校 → 细节打磨，降低AI检测率至30%以下
- **个人素材库**：用真实经历、观点、案例替代AI腔
- **Think Aloud**：透明化思考过程，让用户有掌控感

## 核心理念

> **流程是指南，不是教条；核心原则不可妥协。**

- 流程可以灵活调整（根据任务复杂度和用户需求）
- 核心原则不能妥协（准确性、真实性、透明性）

## 快速开始

### 1. 准备工作

创建以下文件夹结构：

```bash
mkdir -p _briefs _knowledge_base _协作文档 写作参考 个人素材库/主题索引 images
```

### 2. 准备个人素材库（重要！）

这是降AI味的核心！

**方法A：导出即刻动态（推荐）**
- 导出你的即刻动态到CSV文件
- 保存为 `个人素材库/全部即刻动态.csv`

**方法B：整理主题素材**
- 按主题整理你的真实经历、观点、案例
- 保存到 `个人素材库/主题索引/` 文件夹

### 3. 创建风格指南

创建 `写作参考/风格指南.md`，记录你的写作风格。

详细步骤请查看 [快速开始指南](./QUICKSTART.md)

### 4. 开始使用

**新写作任务**：
```
帮我写一篇关于[主题]的文章，目标读者是[读者]，字数[字数]左右。
```

**修改已有文章**：
```
帮我把这篇文章改得更口语化一些。
```

**文章审校**：
```
帮我审校这篇文章，降低AI味。
```

## 核心流程

### 两层判断机制

**第一层：工作区判断**
- 公众号写作
- 视频创作
- Prompt梳理
- 其他

**第二层：任务类型判断**
- A. 新写作任务（有完整brief）
- B. 新写作任务（无brief只有需求）
- C. 修改已有文章
- D. 文章审校/降AI味
- E. 快速咨询

### 9步完整流程

1. **理解需求 & 保存Brief**
2. **信息搜索与知识管理** ⭐
3. **选题讨论** ⭐ 必做
4. **创建协作文档**（如需测试/配图）
5. **学习用户风格**
6. **等待用户提供测试数据**
7. **创作初稿**
8. **三遍审校（降AI味）** ⭐
9. **文章配图** ⭐

详细说明请查看 [SKILL.md](./SKILL.md)

### 7个关键要点

1. **Think Aloud透明化思考**：每次决策都说明思考过程
2. **调研先行**：涉及新技术必须搜索最新信息
3. **选题讨论必做**：不要直接写文章，先讨论选题
4. **个人素材库降AI味**：用真实经历、观点、案例
5. **三遍审校机制**：系统化降低AI检测率
6. **文章配图流程**：直接完成配图，不只写指南
7. **协作文档**：明确分工，让用户知道需要配合什么

## 核心原则（不可妥协）

- ❌ 绝不编造数据
- ❌ 绝不使用过时信息
- ❌ 绝不省略Think Aloud
- ❌ 绝不跳过用户确认（重要决策）

## 实际效果

使用这个工作流写出的文章：
- 小红书数据不错，引起讨论
- 没有任何一个人怀疑是AI写的
- AI检测率低于30%

**为什么效果好？**
- 文章有真实案例（从个人素材库提取）
- 语言风格像本人（三遍审校去掉AI腔）
- 观点和态度明确（不是中立客观的AI腔）
- 细节真实（具体数字、具体场景）

## 常见问题

### Q: 为什么要两层判断？
A: 避免AI误判任务类型，确保走正确的流程。

### Q: 为什么选题讨论是必做的？
A: 避免方向错误造成的大量返工，让用户有掌控感。

### Q: 如何降低AI味？
A: 三遍审校 + 个人素材库。删除套话、改成口语化、加入真实细节。

### Q: 个人素材库从哪里来？
A: 从你的即刻动态、历史文章、真实经历中提取。

### Q: 流程可以跳过吗？
A: 流程可以灵活调整，但核心原则不能妥协（准确性、真实性、透明性）。

### Q: 需要多长时间才能上手？
A: 如果只是基本使用，5分钟就能上手。如果要用好，需要1-2周的磨合期。

更多问题请查看 [快速开始指南](./QUICKSTART.md)
