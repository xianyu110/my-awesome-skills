## 核心信息

### 什么是 Banana/Nano Banana Pro?

**Banana** = **Nano Banana Pro** = **Google Gemini 3 Pro Image** 模型

- **官方名称**: Gemini 3 Pro Image
- **发布时间**: 2025年11月
- **开发商**: Google
- **定位**: AI驱动的科研绘图和信息图表生成工具

## 技术能力

### 核心优势
1. **文本渲染**: 97%+ 多语言文字准确率
2. **图像一致性**: 表现优于同类AI工具
3. **可编辑性**: 支持SVG可视化编辑
4. **科学准确性**: 在生物医学领域表现突出

### 输出格式
- HTML (Tailwind, GSAP, Three.js, Echarts)
- Canvas
- SVG
- 支持视觉编辑和调试控制台

## 与 BioRender 对比

### 价格
- **BioRender**: $15/月 (学生价)
- **Nano Banana Pro**: 通过 Google Vertex AI 可免费使用 (有使用限制)

## 应用场景

### 科研绘图类型
1. **学术论文插图** - 30秒生成NeurIPS级插图
2. **技术路线图** - 项目申请书、开题报告
3. **实验流程图** - Method部分配图
4. **学术海报** - 会议展示
5. **机制图** - 生物医学通路图
6. **数学图表** - 自动生成准确数学图像

### 适用人群
- 科研人员
- 研究生(硕士/博士)
- 需要绘制科研图表的实验室成员

## 使用方式

### 访问途径
1. **gemini.google.com** - 官方入口
   - 点击 "Tools" → "Image Creation"
   - 有每日使用限制(50-100张图后会降级)

2. **国内镜像**
   - https://geminiai.asia (无需魔法)
   - 需注册账号,选择节点

### 使用技巧
- 用英文关键词搜索素材(Centrifuge, Mouse, Cell culture等)
- 提供清晰的Prompt描述
- 分步骤生成复杂图表
- 人工审核科学准确性
