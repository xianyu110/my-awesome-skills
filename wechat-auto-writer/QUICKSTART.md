# 快速开始指南

## 5分钟快速上手

### 第一步：创建文件夹结构

在你的工作区创建以下文件夹：

```bash
mkdir -p _briefs _knowledge_base _协作文档 写作参考 个人素材库/主题索引 images
```

### 第二步：准备个人素材库（重要！）

这是降AI味的核心！

**方法A：导出即刻动态（推荐）**

1. 登录即刻网页版
2. 导出你的所有动态到CSV文件
3. 保存为 `个人素材库/全部即刻动态.csv`

**方法B：手动整理素材**

在 `个人素材库/主题索引/` 文件夹中，按主题创建文件：

```
个人素材库/主题索引/
├── AI编程工具.md
├── 产品开发.md
├── 写作经验.md
└── ...
```

每个文件记录你在这个主题下的：
- 真实经历
- 真实观点
- 真实案例

### 第三步：创建风格指南

创建 `写作参考/风格指南.md`，记录你的写作风格：

```markdown
# 我的写作风格

## 开头方式
- 用真实经历引入
- 直接抛出问题
- 用数据或案例开场

## 结构偏好
- 总-分-总结构
- 每部分有小结
- 重点用加粗或列表突出

## 语言特征
- 口语化，多用短句
- 多用"你"、"我"
- 少用书面词汇
- 多用具体数字和案例

## 金句风格
- 简洁有力
- 有态度
- 不超过20字
```

### 第四步：开始使用

现在你可以开始使用了！

**新写作任务**：
```
帮我写一篇关于[主题]的文章，目标读者是[读者]，字数[字数]左右。
```

**修改已有文章**：
```
帮我把这篇文章改得更口语化一些。

（粘贴文章内容）
```

**文章审校**：
```
帮我审校这篇文章，降低AI味。

（粘贴文章内容）
```

## 第一次使用示例

### 场景：写一篇关于Claude Code的文章

**你**：
```
帮我写一篇关于Claude Code的文章。

主题：Claude Code vs Cursor对比
目标读者：开发者
字数：3000字左右
需要真实测试数据
```

**AI会做什么**：

1. **保存Brief**
   - 创建 `_briefs/claude-code-商单brief.md`

2. **信息搜索**
   - 搜索Claude Code的最新信息
   - 保存到 `_knowledge_base/claude-code-2025-01.md`

3. **选题讨论**
   - 提供3-4个选题方向
   - 每个包含标题、角度、大纲
   - **等你选择**

4. **创建协作文档**
   - 创建 `_协作文档/claude-code-对比-协作文档.md`
   - 列出测试任务清单
   - 列出配图需求清单

5. **等待你提供测试数据**
   - 你完成测试后，把数据发给AI

6. **学习你的风格**
   - 阅读 `写作参考/风格指南.md`
   - 阅读你的历史文章

7. **使用个人素材库**
   - 在 `个人素材库/全部即刻动态.csv` 中搜索相关内容
   - 提取你的真实观点和经历

8. **创作初稿**
   - 基于真实数据写作
   - 融入你的风格和素材

9. **三遍审校**
   - 内容审校（逻辑、事实、结构）
   - 风格审校（去AI味）
   - 细节打磨（标点、排版、节奏）

10. **文章配图**
    - 创建 `images/claude-code-对比/` 文件夹
    - 获取/生成图片
    - 插入Markdown

## 常见使用场景

**你**：


- 跳过测试环节
- 基于已有信息快速写作
- 仍然会做选题讨论和三遍审校

**你**：
```
帮我把这篇文章改得更口语化。

```

**AI会**：
- 直接进入修改流程
- 重点做风格审校

**你**：

- 直接进入三遍审校流程
- 详细说明发现的问题
- 逐一修改


## 进阶使用

### 自定义工作区

你可以为不同类型的内容创建不同的工作区：

```
工作区/
├── 公众号写作/
│   ├── _briefs/
│   ├── _knowledge_base/
│   ├── ...
├── 视频创作/
│   ├── _briefs/
│   ├── _scripts/
│   ├── ...
└── Prompt梳理/
    ├── _prompts/
    ├── _examples/
    ├── ...
```

每个工作区可以有自己的规则和流程。

### 积累个人素材库

这是长期工作，但非常值得：

1. **定期整理**
   - 每周整理一次即刻动态
   - 每月整理一次主题素材

2. **分类存储**
   - 按主题分类
   - 按时间分类
   - 按类型分类（经历/观点/案例）

3. **持续更新**
   - 有新的经历就记录
   - 有新的观点就记录
   - 有新的案例就记录

### 优化风格指南

随着使用，不断优化你的风格指南：

1. **记录好的表达**
   - 你喜欢的开头方式
   - 你喜欢的金句
   - 你喜欢的结构

2. **记录不好的表达**
   - 你不喜欢的AI腔
   - 你不喜欢的套话
   - 你不喜欢的句式

3. **定期更新**
   - 每写完一篇文章，回顾一下
   - 哪些地方满意，哪些地方不满意
   - 更新到风格指南中

## 常见问题

### Q: 我没有即刻账号，怎么办？
A: 可以手动整理素材。把你的真实经历、观点、案例记录到 `个人素材库/主题索引/` 文件夹中。

### Q: 我的素材很少，怎么办？
A: 没关系，从现在开始积累。每次有新的经历、观点、案例，就记录下来。慢慢积累，素材会越来越多。

### Q: 我不知道怎么写风格指南，怎么办？
A: 可以先用默认的风格指南，然后在使用过程中慢慢调整。或者参考你喜欢的作者的风格。

### Q: 我可以跳过某些步骤吗？
A: 可以。流程是灵活的，可以根据任务复杂度调整。但核心原则不能妥协（准确性、真实性、透明性）。

### Q: 如何判断AI味是否降低了？
A: 可以用AI检测工具测试，目标是30%以下。或者让朋友读一读，看是否像你本人在说话。

### Q: 需要多长时间才能上手？
A: 如果只是基本使用，5分钟就能上手。如果要用好，需要1-2周的磨合期，积累素材和优化风格指南。
