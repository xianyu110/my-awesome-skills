# 书稿写作体例规范助手（清华大学出版社）

## ⚠️ 使用前必读

读取 `checklist.md` 进行交稿前全面自查，读取 `style-guide.md` 查询具体格式规则。

---

## 文件索引

| 文件 | 内容 |
|------|------|
| `style-guide.md` | 完整排版规范：标题、正文、插图、插表、代码、公式 |
| `checklist.md` | 交稿前自查清单 |

---

## 场景路由

| 场景 | 操作 |
|------|------|
| 检查书稿格式 | 读 `checklist.md` 逐项核对 |
| 查询标题格式 | 读 `style-guide.md` → 标题层级章节 |
| 查询插图规范 | 读 `style-guide.md` → 插图章节 |
| 查询代码格式 | 读 `style-guide.md` → 代码章节 |
| 查询公式规范 | 读 `style-guide.md` → 公式章节 |
| 交稿前检查 | 读 `checklist.md` 全部通过后提交 |

---

## 快速参考

### 标题层级对照

| 级别 | 格式 | 字体 | 示例 |
|------|------|------|------|
| 一级标题 | 居中 | 三号黑体 | `第1章 标题` |
| 二级标题 | 顶格 | 小三号黑体 | `1.1 标题` |
| 三级标题 | 顶格 | 四号黑体 | `1.1.1 标题` |
| 四级标题 | 前空两格 | 小四号黑体 | `1. 标题` |
| 五级标题 | 五号黑体 | 无标点 | `1）标题` |

### 一票否决项

- 图题写在图上方（正确：图下方）
- 表题写在表下方（正确：表上方）
- 代码未使用代码清单格式
- 英文单词加了双引号（如："Next"→ Next）
- 多于5行的代码未在开头注明文件名
