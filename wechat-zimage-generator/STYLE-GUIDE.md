# 配图风格指南

> 参考 baoyu-article-illustrator 的9种专业风格，优化公众号配图质量

## 🎨 九种专业风格

### 1. notion (默认推荐) ⭐

**特点**: 极简手绘线条，知识分享风格

**色彩方案**:
- 主色: 黑色 (#1A1A1A)、深灰 (#4A4A4A)
- 背景: 纯白 (#FFFFFF)
- 点缀: 浅蓝 (#A8D4F0)、浅黄 (#F9E79F)、浅粉 (#FADBD8)

**视觉元素**:
- 简单线条涂鸦
- 手绘摇摆效果
- 几何图形
- 火柴人
- 最大化留白

**适用场景**: 知识分享、概念解释、SaaS内容、生产力文章

**提示词模板**:
```
[主题]，notion风格，极简手绘线条，黑色线条白色背景，浅蓝浅黄点缀，几何图形，火柴人，最大化留白，手绘摇摆效果，16:9横版
```

- 精致线条

### 6. minimal (极简禅意)

**特点**: 超级干净、禅意、专注

**色彩方案**:
- 主色: 纯黑 (#000000)、白色 (#FFFFFF)
- 背景: 白色或灰白 (#FAFAFA)
- 点缀: 单一颜色（从内容衍生）

**视觉元素**:
- 单一焦点
- 最大化负空间
- 细而精确的线条

**适用场景**: 哲学、极简主义、专注解释

**提示词模板**:
```
[主题]，minimal极简风格，纯黑纯白，白色灰白背景，单一焦点最大化留白，细而精确线条，禅意专注，16:9横版
```

### 9. sketch (草图手稿)

**特点**: 原始、真实、笔记本风格

**色彩方案**:
- 主色: 铅笔灰 (#4A5568)、纸白 (#FAFAFA)
- 背景: 灰白纸张纹理 (#F7FAFC)
- 点缀: 单一高光色（蓝、红或黄）

**视觉元素**:
- 粗糙草图线条
- 箭头
- 手写标签
- 划掉的标记

**适用场景**: 进行中的想法、头脑风暴、思考过程

**提示词模板**:
```
[主题]，sketch草图风格，铅笔灰纸白，灰白纸张纹理背景，粗糙草图线条箭头，手写标签划掉标记，原始真实，16:9横版
```

## 🎯 风格选择指南

### 按内容类型选择

| 内容类型 | 推荐风格 | 备选风格 |
|---------|---------|---------|
| 知识分享、概念解释 | notion | minimal, elegant |
| AI、编程、技术 | tech | notion, sketch |
| 个人成长、生活 | warm | playful, nature |
| 商业、专业 | elegant | notion, minimal |
| 观点、警告 | bold | tech, elegant |
| 教程、指南 | playful | warm, notion |
| 环保、健康 | nature | warm, minimal |
| 创意、头脑风暴 | sketch | playful, notion |

| 情感基调 | 推荐风格 |

| 专业严肃 | elegant, minimal |

| 简洁专注 | minimal, notion |

## 📝 提示词优化技巧

### 1. 基础结构

```
[主题] + [风格] + [色彩] + [背景] + [元素] + [特征] + [比例]
```

### 2. 必备元素

- ✅ 明确主题
- ✅ 指定风格
- ✅ 色彩方案
- ✅ 视觉元素
- ✅ 16:9比例

### 3. 质量关键词

添加这些词提升质量：
- "高质量"
- "专业设计"
- "清晰明亮"
- "精美细节"
- "手绘风格"

### 4. 避免的词汇

- ❌ "好看的"（太模糊）
- ❌ "漂亮"（主观）
- ❌ "图片"（废话）
- ❌ 过长的描述（>100字）

## 🔄 风格组合建议

### 同一文章多张图

**方案1: 单一风格**
- 所有配图使用同一风格
- 保持视觉一致性
- 适合大多数情况

**方案2: 渐进风格**
- 封面: bold/elegant（吸引注意）
- 正文: notion/minimal（易读）
- 结尾: warm/playful（温馨）

**方案3: 对比风格**
- 问题部分: bold（强调）
- 解决方案: warm（友好）
- 总结: notion（清晰）

## 🎨 色彩搭配速查

### 科技感
- 蓝+紫+青: 专业科技
- 黑+蓝+白: 现代简约
- 深蓝+霓虹: 未来感

### 温暖感
- 橙+黄+棕: 温馨友好
- 粉+米+金: 柔和舒适
- 红+橙+黄: 活力热情

### 专业感
- 蓝灰+白: 商务专业
- 黑+金+白: 高端奢华
- 深蓝+灰: 稳重权威

### 自然感
- 绿+棕+米: 大地自然
- 蓝+绿+白: 清新自然
- 绿+黄+白: 生机活力
