# 字流多平台内容分发 - Claude Skill

## 简介

这是一个帮助内容创作者实现"写一次内容，适配多平台发布"的 Claude Skill。

基于字流2.0产品的核心理念：让创作者专注于内容创作本身，把繁琐的平台适配、格式转换、字段生成交给AI。

## 核心能力

### 1. 多平台内容适配

从一篇长文自动生成14个平台的适配内容：

**长文平台（4个）**：
- 微信公众号
- 知乎
- 掘金
- 知识星球

**短图文平台（5个）**：
- 小绿书
- 小红书图文
- 微博
- 即刻
- Twitter

**视频平台（5个）**：
- 视频号
- 抖音
- 小红书视频
- B站
- YouTube

### 2. 智能字段生成

为每个平台自动生成：
- 适配的标题（符合平台风格）
- 优化的描述（符合字数限制）
- 精准的标签（提高曝光率）
- 合适的封面（符合尺寸要求）
- 专属的口播稿（视频平台）

### 3. 平台规则遵守

自动处理不同平台的规则差异：
- 内容规则（联系方式、外链、二维码）
- 格式要求（Markdown、富文本、纯文本）
- 字数限制（280字符到无限制）
- 图片数量（4张到无限制）

### 4. 内容合规检查

自动检查：
- 敏感词
- 格式错误
- 规则违反
- 字数超限

## 核心原则

### ✅ 必须遵守

1. **内容一致性**：核心信息不变，只调整表达方式
2. **平台合规**：严格遵守各平台规则
3. **真实性**：不夸大，不编造数据
4. **用户视角**：保持原作者的语气和观点

### ❌ 绝不允许

1. **违规内容**：不生成违反平台规则的内容
2. **过度改写**：不改变原文核心意思
3. **标题党**：不使用虚假夸张的标题
4. **抄袭**：不使用他人的原创表达

## 常见问题

### Q1: 生成的内容需要人工审核吗？

A: 建议审核。虽然AI会自动检查合规性，但最终发布前建议人工检查：
- 内容准确性
- 语气是否符合个人风格
- 平台规则是否完全遵守

### Q2: 可以自定义平台规则吗？

A: 可以。如果你对某个平台有特殊要求，可以在使用时明确说明。

### Q3: 支持批量处理吗？

A: 支持。可以一次性处理多篇文章，生成完整的内容分发包。

### Q4: 生成的内容会重复吗？

A: 不会。每个平台的内容都会根据平台特点进行适配，保持差异化。

### Q5: 如何保证内容质量？

A: 
1. 基于原文核心信息
2. 遵循平台最佳实践
3. 保持作者语气和观点
4. 建议人工最终审核
