**【核心概念：通俗化降维解释】**
二、什么是 Skills？
简单说，Skills 就是给 Claude/Codex 装的“专业技能包”。
就像手机上装 App 一样，每个 Skill 是一个专门功能。
比如有专门写文档的、有专门做代码审查的、有专门帮你头脑风暴的。
技术原理就是通过结构化的提示词模板，让 AI 知道在什么场景该怎么做。
这部分已经有很多人分享过了，这里就不展开了。
重点是：怎么用上这些 Skills，让你的 Codex 真正变聪明。

**【价值抛售：三个必用理由】**
三、为什么要用 Skills？

1. 让 Codex 具备专业领域知识
每个 Skill 都包含了特定领域的最佳实践和工作流程。

* 没有 Skills 的时候，Codex 只能根据通用知识来帮你，你需要一步步告诉它该怎么做。
* 有了 Skills 之后，它已经“学会”了各种专业领域的正确做法——写文档有文档的规范，写代码有代码的流程，做设计有设计的方法。

2. 自动选择合适的工具
这是最省心的地方。下载 Skills 后，你不用记住什么时候该用什么 Skill，Codex 会根据任务自动判断和调用。
就像工具箱里有了各种专业工具，Codex 知道什么情况下该拿哪个。
3. 输出质量大幅提升
因为每个 Skill 内置了专业的工作流程和检查机制，最终输出的结果更专业、更符合预期。
这就是为什么用了 Skills 后，感觉 Codex “从小学生升级到大学生”——它不再是通用助手，而是各个领域的专家。

**【保姆级教程：手把手实操】**
四、怎么用 Skills？（手把手教程）
Step 1：去 skillsmp 网站下载 Skills
打开 skillsmp 网站 (skillsmp[.]c/o/m)
这里收录了 44,274 个来自 GitHub 的开源 Agent Skills。

Step 2：选择并下载 Skills
我的建议：如果你不知道选哪个，选择 Claude 官方的进行下载，多星的进行下载。
点击任意一个 Skill 进入详情页，可以看到：Skill 的功能说明、使用场景、下载按钮。

Step 3：安装 Skills（超简单）
下载后会得到一个 `.zip` 文件，解压后是一个文件夹（包含 `SKILL.md` 文件）。
安装方法（小白友好）：

1. 解压下载的 zip 文件
2. 把解压出来的文件夹直接拖入：
macOS / Linux 系统：

* Codex: `~/.codex/skills/` 文件夹
* Claude: `~/.claude/skills/` 文件夹
Windows 系统：
* Codex: `C:\Users\<你的用户名>\.codex\skills\`
* Claude: `C:\Users\<你的用户名>\.claude\skills\`
就这么简单！不需要运行任何命令。
（配合画面：将解压后的文件夹拖入 Skills 文件夹）

Step 4：验证 Skills 是否生效
安装后，需要验证一下是否成功加载。
首先重启 Codex 或 Claude 然后输入命令：
`/skills`
如果成功，会显示已加载的 Skills 列表和 token 数。
看到列表说明成功了！🎉

**【进阶拔高：差异化玩法】**
五、进阶玩法
自己创建 Skill
除了下载现成的，你还可以自己创建 Skill。
适合场景：公司特定的开发流程、团队规范、或者你自己总结的最佳实践。
基本结构很简单：

* 一个 Skill = 一个文件夹 + `SKILL.md` 文件
* 文件里包含：名称、描述、什么时候用、怎么用

具体怎么写可以参考官方文档或 GitHub 上的开源示例。
Codex文档：developers[.]openai[.]com/codex/skills
Claude文档：code[.]claude[.]com/docs/en/skills

用 superpowers 统一调度
我自己用的方法是：下载了 100+ Skills 后，用 superpowers 来统一管理。
superpowers 是一个完整的开发工作流系统，包含头脑风暴、Git 管理、计划、执行、TDD、代码审查等 7 个阶段。它会帮你自动调度所有 Skills，你只需要描述需求就行。
在 GitHub 搜索 obra/superpowers 可以找到。
感兴趣的可以去研究一下，这是更高级的玩法。

# Part 2：结构拆解

## 1. 框架结构

* **黄金开篇**：用“两个世界”制造焦虑差，用“大学生版”抛出诱饵。
* **概念降维**：用“手机装 App”类比复杂的技术插件，降低认知门槛。
* **价值论证**：通过“专业度、自动化、质量”三点对比，强化使用必要性。
* **极简实操**：分步骤教学，强调“无需命令、直接拖拽”，消除技术恐惧。
* **高阶拓展**：提供“自定义”和“统一调度”的高端玩法，筛选硬核用户。
* **结尾促动**：简单的指令性结尾，引导立即行动。

## 2. 为什么有效

* **极强的获得感**：将复杂的代码环境配置简化为“下载-拖拽”，小白也能觉得自己变强了。
* **通俗易懂的类比**：把 AI Skills 比作手机 App，瞬间讲清了技术原理，受众面极广。

## 3. 心理触发点

* **情绪**：**FOMO（错失恐惧）**——“用不用处在两个世界”；**成就感**——“从小学生升级到大学生”。
* **动机**：**省力原则**——强调“自动选择工具”、“不需要运行命令”。
* **认知偏差**：**锚定效应**——先设定“AI 只是小学生”的锚点，再抛出“大学生”的升级预期。

## 4. 复刻所需背景

* **核心知识**：熟悉 Cursor/VSCode/Claude 等 AI 编程工具的基本配置和文件路径。
* **素材准备**：需要录制/截图文件路径操作界面、前后效果对比图、资源网站录屏。
* **背书资源**：最好准备 1-2 个具体的 Skill 案例（如前端设计、Git 管理）作为演示。
