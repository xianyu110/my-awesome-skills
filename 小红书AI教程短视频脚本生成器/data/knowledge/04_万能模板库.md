# 万能模板库

## 使用说明

本模板库汇总了27个爆款案例中提炼出的可复用模板,按内容类型分类。每个模板包含:
- 完整的结构框架
- 每个环节的填空说明
- 2-3个实际应用案例
- 适用场景和注意事项

## 七、模板使用指南

### 7.1 如何选择模板

**决策树**:

```
你的内容目标是什么?
│
├─ 教别人用工具 → 工具教程类模板
├─ 帮别人选工具 → 产品测评类模板
├─ 分享经验方法 → 经验分享类模板
├─ 推荐资源合集 → 资源推荐类模板
├─ 分析案例产品 → 案例分析类模板
└─ 输出观点看法 → 观点输出类模板
```

### 7.2 模板改编技巧

**基本原则**:
1. **保留结构**: 框架是经过验证的,不要轻易改动
2. **替换内容**: 把[ ]中的内容替换成你的实际内容
3. **调整顺序**: 根据你的内容特点微调环节顺序
4. **添加个性**: 在框架基础上加入你的个人风格

**举例**:

原模板:
```
【开头Hook】
如果我是[身份]从零开始[做某事],我会这么做。
```

你的改编:
```
【开头Hook】
如果我是文科生从零学Python,我会这么学。
→ 保留"如果我是...从零...我会..."的句式
→ 替换成你的实际身份和目标
```

### 7.3 模板组合使用

**组合1: 教程+避坑**
```
主体用"工具教程模板"
在每个步骤后加入"避坑提示"

```

**组合2: 测评+推荐**
```
主体用"对比测评模板"

效果: 从测评到行动的完整链路
```

**组合3: 经验+案例**

**调整要点**:
- 前3秒必须有视觉冲击
- 全程字幕,无字幕必死
- 节奏快,每15秒一个点
- BGM要配合情绪

**时长控制**:
- 开头: 3秒
- 每个要点: 15-20秒
- 结尾: 5秒

## 十、模板常见问题

**Q1: 用模板会不会没创意?**
A: 模板是框架不是全部。好的内容=成熟框架+独特内容+个人风格

**Q2: 模板需要完全照搬吗?**
A: 不需要。根据你的实际内容灵活调整,模板是参考不是限制。

**Q3: 同类型内容能重复用同一个模板吗?**
A: 可以,但要变化表达方式。同样的结构,不同的案例和语言。

**Q4: 如何判断我用对模板了?**
A: 看数据。如果完播率、互动率、转化率都不错,说明模板选对了。

**Q5: 模板过时了怎么办?**
A: 持续观察爆款内容,更新你的模板库。建议每季度更新一次。
