## 结构一：教程型（Tutorial Flow）

### 流向图

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  开头Hook   │ →  │  痛点放大   │ →  │  方案介绍   │ →  │  分步演示   │ →  │  行动号召   │
│  (5-10秒)   │    │  (15-30秒)  │    │  (10-20秒)  │    │  (主体部分)  │    │  (结尾)     │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
      ↓                  ↓                  ↓                  ↓                  ↓
   抓注意力           共鸣+焦虑          给希望             建信任            促行动
```

### 核心公式

**Hook** → **Problem** → **Solution Preview** → **Step-by-Step Demo** → **CTA**

### 适用场景

- AI工具安装/配置教程
- Web Coding项目实操
- 自动化工作流搭建
- 效率工具使用指南

### 关键要素

| 环节 | 核心任务 | 时长占比 | 情绪曲线 |
|------|----------|----------|----------|
| 开头Hook | 展示成果/抛出痛点 | 5% | 好奇峰值 |
| 痛点放大 | 描述具体场景+共鸣 | 15% | 焦虑上升 |
| 方案介绍 | 承诺"简单/快速/免费" | 10% | 希望点燃 |
| 分步演示 | 1-2-3-4步骤拆解 | 60% | 信任建立 |
| 行动号召 | 评论互动+预告 | 10% | 行动冲动 |

### 万能开头模板

```
"做了一个[解决XX痛点]的工具，全程大白话，没写一行代码，X分钟搞定"
"[XX任务]真的太费时间了，所以我用[AI工具]做了一个自动化的方案"
"[AI工具]到底要怎么安装？怎么登录？今天一次讲清楚"
```

## 结构二：洞察型（Insight Flow）

### 流向图

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  反常识Hook │ →  │  认知破除   │ →  │  框架阐述   │ →  │  案例验证   │ →  │  升华总结   │
│  (挑战认知)  │    │  (打破旧见)  │    │  (核心模型)  │    │  (具体证据)  │    │  (金句收尾)  │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
      ↓                  ↓                  ↓                  ↓                  ↓
   制造冲突           建立权威          提供框架           增强信任          强化记忆
```

### 核心公式

**Contrarian Hook** → **Myth Busting** → **Framework** → **Evidence** → **Punch Line**

### 适用场景

- AI时代职业发展观点
- 个人成长方法论
- 行业趋势解读
- 思维模式升级

### 关键要素

| 环节 | 核心任务 | 心理触发 | 关键技巧 |
|------|----------|----------|----------|
| 反常识Hook | 挑战普遍认知 | 好奇+震惊 | "大家都以为...其实..." |
| 认知破除 | 指出误区来源 | 认同+反思 | 对比被动vs主动心态 |
| 框架阐述 | 提供思考模型 | 顿悟+启发 | 用图/公式可视化 |
| 案例验证 | 权威背书+个人经历 | 信任+共鸣 | 硅谷案例/亲测数据 |
| 升华总结 | 一句话金句 | 记忆+传播 | 英文+中文双语金句 |

### 万能开头模板

```
"硅谷现在很流行的一个做法叫[XX]，但国内一提到这个大家第一反应是[担忧]"
"我觉得[某品质]是AI时代最重要的能力，但很多人完全搞反了"
"大家不管做什么，都要记住一个法则：[核心洞察]"
```

## 结构三：经验分享型（Story Flow）

### 流向图

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  个人Hook   │ →  │  背景铺垫   │ →  │  方法展开   │ →  │  效果验证   │ →  │  价值传递   │
│  (我是谁)   │    │  (为何分享)  │    │  (怎么做)   │    │  (做到了)   │    │  (你也可以)  │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
      ↓                  ↓                  ↓                  ↓                  ↓
   建立人设           制造信任          核心价值           证明可行           激励行动
```

### 核心公式

**Personal Hook** → **Context** → **Method** → **Result** → **Empowerment**

### 适用场景

- 学习心得分享
- 年度复盘总结
- 从0到1经历
- 转型/成长故事

### 关键要素

| 环节 | 核心任务 | 信任构建 | 话术要点 |
|------|----------|----------|----------|
| 个人Hook | 亮出身份/成果 | 资质背书 | "作为一个文科生/小白..." |
| 背景铺垫 | 说明契机/动机 | 真实性 | "去年我还在困惑..." |
| 方法展开 | 分点阐述 | 系统性 | "第一步...第二步..." |
| 效果验证 | 数据+感受 | 可信度 | "粉丝从X涨到Y..." |
| 价值传递 | 赋能观众 | 共情 | "你也可以..." |

### 万能开头模板

```
"[年份]对我来说是[关键词]的一年，我把很多想法都付诸行动了"
"经常有朋友问我怎么有时间做这么多事，今天分享一个技巧"
"如果我是一个[身份]从零开始学[技能]，我会这么学"
```

## 通用Hook公式库

### 痛点型
- "我发现[XX任务]真的太[负面词]了"
- "[场景]的时候你是不是第一反应就是[低效行为]？"

### 成果型
- "用[工具]做了一个[成果]，全程只用了[时间]"
- "我只花了[时间]就完成了[任务]从学习到实践的全过程"

### 反常识型
- "大家都以为[A]，但其实[B]才是关键"
- "这个想法在国内一提出来，第一反应都是[担忧]，但其实..."

### 权威型
- "硅谷现在讨论最多的一个词叫[概念]"
- "[大佬]说过一句话让我印象深刻"

### 数据型
- "从[起点数据]涨到[终点数据]，我做对了这几件事"
- "每天[X]分钟，坚持[Y]个月后..."
