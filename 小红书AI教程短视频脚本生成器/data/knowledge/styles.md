## 一、人设标签矩阵

### 核心人设类型

| 人设类型 | 标签关键词 | 信任基础 | 适用内容 |
|----------|------------|----------|----------|
| **技术小白转型者** | 文科生、非技术背景、从零开始 | 同理心+可复制性 | 入门教程、工具测评 |
| **AI学习爱好者** | 业余爱好、持续学习、分享心得 | 真实性+热情 | 学习方法、资源整理 |
| **硅谷观察者** | 湾区、一手信息、follow builders | 权威性+稀缺性 | 行业洞察、趋势解读 |
| **产品实践者** | build in public、side project | 实战经验+可信度 | 产品分享、创业心得 |
| **效率极客** | 自动化、workflow、生产力 | 专业性+实用性 | 工具教程、效率方法 |

### 人设定位公式

```
我是一个 [背景身份] + 从零开始 [学习/做] [领域] + [时间周期] 后 [取得成果]
```

**示例：**
- "我是一个没有技术背景的文科生，从零开始学AI，一年后全网10万粉丝"
- "我是一个产品经理，业余时间用AI做了5个side project"

### 人设强化话术

| 场景 | 话术模板 | 效果 |
|------|----------|------|
| 降低门槛 | "我真的是文科生，这些事情我也就是搞一搞就OK了" | 消除恐惧 |
| 建立共情 | "去年我也是这样的状态，每天很焦虑" | 引发共鸣 |
| 展示真实 | "我踩过的坑分享给大家，避免你们走弯路" | 增加信任 |
| 强调业余 | "做内容只是我业余的一个爱好" | 降低功利感 |

## 二、高频口头禅词库

### A类：降低门槛（最高频）

| 口头禅 | 使用场景 | 心理作用 |
|--------|----------|----------|
| **大白话** | 介绍方法/工具时 | 消除专业壁垒 |
| **很简单** | 步骤讲解前 | 降低学习焦虑 |
| **X分钟搞定** | 标题/开头 | 强调效率 |
| **0代码/不用写代码** | 技术教程 | 消除技术恐惧 |
| **文科生也能** | 全程适用 | 扩大受众范围 |
| **手把手教你** | 教程类内容 | 承诺陪伴感 |

### B类：强调价值（高频）

| 口头禅 | 使用场景 | 心理作用 |
|--------|----------|----------|
| **亲测有效** | 推荐工具/方法 | 增加可信度 |
| **一手信息** | 信息源分享 | 强调稀缺性 |
| **干货** | 内容介绍 | 提升价值感 |
| **这个很重要** | 强调关键点 | 引导注意力 |
| **我自己每天都在用** | 工具推荐 | 真实背书 |

### C类：制造紧迫（中频）

| 口头禅 | 使用场景 | 心理作用 |
|--------|----------|----------|
| **AI时代** | 趋势类内容 | 制造时代感 |
| **不能错过** | 资源推荐 | FOMO触发 |
| **越来越重要** | 观点输出 | 强调必要性 |
| **现在是最好的时机** | 行动号召 | 紧迫感 |

### D类：建立连接（常用）

| 口头禅 | 使用场景 | 心理作用 |
|--------|----------|----------|
| **很多朋友问我** | 引入话题 | 证明需求真实 |
| **分享给大家** | 内容过渡 | 利他姿态 |
| **希望对你有帮助** | 结尾 | 真诚收尾 |
| **评论区扣X** | 互动引导 | 促进参与 |

## 三、情绪词密度指南

### 情绪词分类

#### 正向情绪词（占比60%）

| 类型 | 词汇 | 使用时机 |
|------|------|----------|
| **希望** | 可能、机会、潜力、未来 | 方案介绍时 |
| **兴奋** | 惊艳、太酷了、非常不错 | 展示成果时 |
| **赋能** | 你也可以、人人都能、无门槛 | 行动号召时 |
| **满足** | 收获很大、值得、物超所值 | 价值强调时 |
| **好奇** | 有趣、独特、没见过 | 内容引入时 |

#### 负向情绪词（占比30%）

| 类型 | 词汇 | 使用时机 |
|------|------|----------|
| **焦虑** | 担心、困惑、不知道该怎么办 | 痛点描述时 |
| **痛苦** | 太难了、费时费力、头痛 | 问题放大时 |
| **恐惧** | 被淘汰、跟不上、错过 | FOMO触发时 |
| **挫败** | 试了很多次、一直失败 | 共情建立时 |

#### 中性过渡词（占比10%）

| 类型 | 词汇 | 使用时机 |
|------|------|----------|
| **转折** | 但是、不过、然而 | 观点转换时 |
| **递进** | 而且、并且、更重要的是 | 层层深入时 |
| **总结** | 所以、因此、总的来说 | 段落收尾时 |

### 情绪曲线设计

```
     情绪强度
        ↑
   高   │      ★成果展示
        │     /    \
        │    /      \   ★行动号召
   中   │   /        \_____
        │  /
        │ /
   低   │★痛点共鸣
        └────────────────────→ 时间轴
         开头    中间    结尾
```

**最佳节奏：**
1. 开头：低谷（痛点共鸣）→ 轻微上升（给出希望）
2. 中间：持续上升（解决方案）→ 高峰（成果展示）
3. 结尾：平稳高位（行动号召）→ 余韵（预告/感谢）

## 四、禁用词清单

### A类：技术黑话（严禁）

| 禁用词 | 替代表达 | 原因 |
|--------|----------|------|
| API调用 | 连接/对接 | 技术门槛高 |
| 部署上线 | 发布/上线 | 过于专业 |
| 数据库 | 存储/保存 | 不够直观 |
| 算法/模型训练 | AI学习/AI理解 | 晦涩难懂 |
| 代码逻辑 | 操作流程 | 劝退新手 |

### B类：过度承诺（避免）

| 禁用词 | 问题 | 建议 |
|--------|------|------|
| 躺赚/暴富 | 违规风险 | 改为"提升效率" |
| 一定能/保证 | 过度承诺 | 改为"可能/大概率" |
| 全网最全/最强 | 夸大其词 | 改为"我认为最好的" |
| 独家/首发 | 难以验证 | 改为"我的分享" |

### C类：负面/争议词（慎用）

| 禁用词 | 问题 | 建议 |
|--------|------|------|
| 韭菜/割韭菜 | 负面联想 | 改为"踩坑/交学费" |
| 抄袭/山寨 | 引发争议 | 改为"借鉴/参考" |
| 垃圾/废物 | 过于负面 | 改为"不太好用" |
| 骗子/坑人 | 极端表达 | 改为"需要注意" |

### D类：距离感词汇（少用）

| 禁用词 | 问题 | 建议 |
|--------|------|------|
| 您/贵 | 过于正式 | 改为"你/大家" |
| 本人/鄙人 | 距离感强 | 改为"我" |
| 诸位/各位 | 书面化 | 改为"大家/朋友们" |

## 五、句式特征

### 高转化句式

| 句式类型 | 模板 | 示例 |
|----------|------|------|
| **设问句** | "[问题]？其实很简单" | "AI怎么学？其实很简单" |
| **假设句** | "如果你[情况]，那你一定要[行动]" | "如果你想学AI，那你一定要看这个" |
| **对比句** | "以前[旧方式]，现在[新方式]" | "以前要写代码，现在大白话就行" |
| **数字句** | "[数字]个[技巧/方法/步骤]" | "3个让你效率翻倍的AI技巧" |
| **反问句** | "你是不是也[痛点]？" | "你是不是也不知道AI怎么学？" |

### 节奏控制

**短句为主（7-15字）**
- "很简单"
- "话不多说，直接开工"
- "这个非常重要"

**长句辅助（20-30字）**
- 解释复杂概念时使用
- 每2-3个短句配1个长句

### 口语化处理

| 书面表达 | 口语化表达 |
|----------|------------|
| 因此我认为 | 所以我觉得 |
| 建议各位 | 建议大家 |
| 相当于 | 就像是/差不多就是 |
| 能够 | 可以/能 |
| 进行 | 做/搞 |
